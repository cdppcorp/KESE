## 2. 적용 범위

### 로봇 유형
| 유형 | 세부 내용 | 보안 특성 |
|------|----------|----------|
| 산업용 로봇 | 제조 공정 자동화, 생산 라인 | 제어 시스템 안정성·안전성 |
| 서비스용 로봇 | 물류, 청소, 안내 등 | 이동성·외부 시스템 연동 보안 |
| 의료용 로봇 | 수술 보조, 재활, 환자 케어 | 데이터 무결성·시스템 신뢰성 |

### 구성요소
- 로봇 단말 (제어기, 센서, 액추에이터)
- 통신 및 네트워크 (내부 버스, 외부 인터페이스)
- 로봇 소프트웨어 (펌웨어, OS, 응용 프로그램)
- 연계 구성요소 (HMI, 원격 제어, 관제 시스템)

### 라이프사이클
설계 → 개발 → 배포 → 운용 → 폐기
