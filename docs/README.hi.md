# KESE - KISA Enhanced Security Evaluation Kit

प्रमुख सूचना एवं संचार अवसंरचना (CII) की कमजोरी विश्लेषण-मूल्यांकन, AI सुरक्षा मूल्यांकन, रोबोट सुरक्षा, अंतरिक्ष सुरक्षा, सुरक्षित कोडिंग गाइड, जीरो ट्रस्ट सुरक्षा मूल्यांकन और SW आपूर्ति श्रृंखला सुरक्षा (SBOM) के लिए Claude Code स्किल प्लगइन।

---

## अवलोकन

KESE (KISA Enhanced Security Evaluation Kit) एक Claude Code प्लगइन है जो KISA (कोरिया इंटरनेट एवं सुरक्षा एजेंसी) दिशानिर्देशों पर आधारित व्यापक सुरक्षा कमजोरी मूल्यांकन क्षमताएँ प्रदान करता है। यह प्रमुख सूचना अवसंरचना (CII), AI सुरक्षा, रोबोट सुरक्षा, अंतरिक्ष सुरक्षा, सुरक्षित कोडिंग और जीरो ट्रस्ट सुरक्षा मूल्यांकनों का समर्थन करता है।
